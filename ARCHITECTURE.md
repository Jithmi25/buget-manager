## ğŸ”„ Data Flow Examples

### 1. User Login Flow

```
1. User enters email/password in Login.jsx
2. Login.jsx calls signInWithEmail() from supabase.js
3. Supabase verifies credentials
4. Returns JWT access token + user data
5. AuthContext stores user state
6. User redirected to /dashboard
7. ProtectedRoute allows access (user is authenticated)
```

### 2. Add Transaction Flow

```
1. User fills form in AddTransaction.jsx
2. Form submitted â†’ calls addTransaction() from api.js
3. api.js gets current user ID
4. Sends INSERT query to Supabase
5. RLS policy checks: auth.uid() = user_id âœ“
6. Data saved to transactions table
7. Success response returned
8. Dashboard refreshes â†’ fetchTransactions() called
9. User sees new transaction in list
```

### 3. View Transactions Flow

```
1. Dashboard.jsx mounts
2. useEffect calls fetchTransactions()
3. fetchTransactions() calls getTransactions() from api.js
4. api.js sends SELECT query to Supabase
5. RLS automatically filters: WHERE user_id = auth.uid()
6. Only user's transactions returned
7. setTransactions(data) updates state
8. TransactionList.jsx renders the data
```
